# 小明助手 Android APK 项目

## How to get APK without coding

1. Create a new GitHub repository.
2. Upload everything in this folder.
3. Open the repository on GitHub.
4. Go to **Actions**.
5. Click **Build APK**.
6. Click **Run workflow**.
7. After it finishes, download **XiaomingAssistant-debug-apk**.
8. Install `app-debug.apk` on the Android phone.

## First launch
Allow:
- Microphone
- Contacts
- Phone

## Commands
- 小明现在几点
- 小明打电话给爸爸
- 小明打电话给张三

## Notes
This APK uses Android's built-in speech recognition and text-to-speech.
Direct calling requires CALL_PHONE permission.
