# 小明语音助手 PWA

## 功能
- 整个屏幕可点击启动语音识别
- 普通话识别：小明现在几点 / 小明帮我弄个5点的闹钟 / 小明帮我弄个5分钟的倒计时
- 普通话语音播报
- 可安装到安卓手机桌面
- 网页内闹钟和倒计时提醒

## 重要限制
普通网页/PWA 不能可靠地长期在手机后台持续监听“小明”，也不能直接设置手机系统闹钟。
要做到真正后台唤醒词和系统闹钟，需要做原生 Android App。

## 安装到爸爸手机
1. 把这些文件上传到 HTTPS 网站，例如 GitHub Pages、Netlify、Vercel，或自己的 HTTPS 服务器。
2. 用爸爸手机的 Chrome 打开网址。
3. 点“安装到手机”，或 Chrome 菜单里的“添加到主屏幕/安装应用”。
4. 第一次使用时允许麦克风和通知。
